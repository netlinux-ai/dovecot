# dovecot

Dovecot IMAP server packaged as a `.deb` for Debian/Ubuntu.

Built from upstream 2.3.21.1 source with mbox support, SSL, PAM, SQLite,
ICU, lz4, zstd, and sodium support.

## Install

```bash
# From the NetLinux repository
sudo apt install dovecot
```

## Build

Builds automatically via GitHub Actions on push to `main`.
